App Description

The android toolkit app is an one stop shop with multiple tools that make the life of the Android user easy. Here are the list of tools and their uses in brief:

Apps Manager – Viewing and killing apps

Hardware Check – Ability to check all device H/W like speakers, camera etc.

Processes’ Info: Lists all the processes that are running

Clear Memory -Clear Cache (SD Card),Clear App Cache,,
Clear Browser Info, Clear call records

File Manager - Display files and folders in root directory
Backup of all downloaded apps

Connectivity App – Toggle connectivity of GPS, bluetooth etc.

Telnet client – Ability to connect to Telnet client
===========================================================

Here is the details description of the tools and their uses:

Apps & Services
===============
Active Apps - Lists all Apps that are currently active.
Running Services - Lists services that are currently running.
All Apps- Lists all Apps that are installed.
Downloaded Apps - Lists Apps that are downloaded.

Hardware Check:
===============
To check whether drivers are installed for the features listed below:

-Speaker
-Camera
-Microphone
-Ringer(Volume controller)
-Monitor status (Display)

Display
=======
Flashlight - To Switch on the flashlight.
Screen Brightness – To adjust the brightness of the screen
Screen Timeout –To modify the screen timeout

Device Info
===========
System Info - Info about the device
System Log Info - Displaying Apps and Device log info.
Disk Info - Displays memory info (sdcard(Audio, Video, Images and other…) and phone memory)

Processes’ Info
===============
Lists all the processes that are running.
Could check the details of the system process like Process id, %CPU usage, current state and memory consumption.
Could check the above said details and also launch it.

Clear Memory
============
-Clear Cache (SD Card)
-Clear App Cache
-Clear Browser Info
-Clear call records

File Manager:
=============
-Display files and folders in root directory
-Details of number of files and subfolders in the current directory will be displayed.
-Backup of all downloaded apps or a specific app could be done. Android tool kit folder would be created in root directory and backup files will be stored in it.

Connectivity
=============
-Mobile networks switch
-Wi-Fi switch
-GPS switch
-Bluetooth switch
-NFC switching

Telnet Client:
==============
With telnet client, you can connect to remote server that supports telnet connectivity and commands could be executed from the device. Also shows remote Server log to users (Bi-direction Communication).
